NOTE: Slurm-web containers are *fully compatible with Kubernetes*. While Slurm-web
does not ship Kubernetes specific installation guide, the published OCI images
on GHCR are standard containers which can be deployed in Kubernetes (or any other
orchestrator that runs OCI images) using the same configuration, secrets, and
networking principles described in the containers and Compose guides.
